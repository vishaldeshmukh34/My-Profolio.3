Tailwind CSS ‚Äì Complete Notes

Tailwind CSS is a utility-first CSS framework used to build beautiful, modern, responsive UI very quickly with predefined classes.

What is Tailwind CSS?

Tailwind provides utility classes like p-4, mt-6, text-center to style elements directly in your HTML.

Why Tailwind CSS?

  • Fast development
  • Highly customizable
  • Mobile Responsive by default
  • No need to write custom CSS
  • Modern utility-first approach

Installation Methods

1. CDN Method

<script src="https://cdn.tailwindcss.com"></script>

2. Using NPM (Recommended)

npm install -D tailwindcss
npx tailwindcss init

Responsive Design

Tailwind uses breakpoints like:

  • md ‚Äì Medium devices

Customizing Tailwind

Customize the framework using tailwind.config.js.

  theme: {
    extend: {
      colors: {
        primary: "#4f46e5",
      }
    }
  }
}

Dark Mode in Tailwind CSS

Tailwind supports dark mode using the dark: prefix. You can enable it using class or media mode.

1. Using the dark: Class

<div class="bg-white text-black dark:bg-gray-900 dark:text-white">
  Dark Mode Enabled
</div>

2. Dark Mode Colors

dark:bg-gray-800
dark:text-gray-200
dark:border-gray-700

Reusable Button Components

Tailwind allows creating elegant UI buttons with utility classes.

1. Primary Button

  Primary Button
</button>

2. Outline Button

<button class="px-4 py-2 border border-blue-600 text-blue-600 rounded-lg hover:bg-blue-600 hover:text-white">
  Outline Button
</button>
