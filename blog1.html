    Interface in Java

    An interface in Java is a blueprint of a class. It contains only abstract methods (until Java 8) and is used to achieve 100% abstraction and multiple inheritance.

    Characteristics:

    • Contains abstract methods (Java 8+ â†’ default & static allowed)
    • Variables are public, static, final by default
    • A class can implement multiple interfaces
    interface Vehicle {
  void start();
}

class Car implements Vehicle {
  public void start() {
    System.out.println("Car starts!");
  }
}

    Type Casting in Java

    Type casting converts one data type into another. Java supports implicit and explicit casting.

    1. Implicit Casting (Widening)

    Smaller â†’ Larger datatype (Automatic)

    int a = 10;
double b = a;  // int â†’ double

    2. Explicit Casting (Narrowing)

    Larger â†’ Smaller datatype (Manual)

    double x = 20.8;
int y = (int) x;  // double â†’ int

    Constructors in Java

    A constructor is a special method used to initialize objects. It has the same name as the class and no return type.

    • Default constructor
    • Parameterized constructor
    • Copy constructor (manual)
    class Student {
  String name;

  Student() {  
    name = "Unknown";  
  }

  Student(String n) {
    name = n;
  }
}

    Inheritance in Java

    Inheritance allows one class to acquire properties of another.

    • Single Inheritance
    • Multilevel Inheritance
    • Hierarchical Inheritance
    class A {
  void displayA() { System.out.println("A"); }
}

class B extends A {
  void displayB() { System.out.println("B"); }
}

    Polymorphism in Java

    Polymorphism means "many forms". It allows the same method to behave differently.

    1. Method Overloading (Compile-time)

    void sum(int a, int b) {}
void sum(double a, double b) {}

    2. Method Overriding (Run-time)

    class A { void show() {} }
class B extends A { void show() {} }

    Abstract Class

    Abstract classes cannot be instantiated. They contain abstract + non-abstract methods. 

    abstract class Animal {
  abstract void sound();
}

class Dog extends Animal {
  void sound() { System.out.println("Bark"); }
}
