    Introduction to JavaScript

    JavaScript is a lightweight, powerful, and high-level programming language that allows developers to make websites interactive. It is one of the core technologies of the web, along with HTML and CSS. JavaScript runs directly inside the browser and is essential for modern web development.

    Why Learn JavaScript?

    • Most demanding skill in Web Development
    • Runs on both frontend and backend (Node.js)
    • Easy to learn and extremely beginner-friendly
    • Used in top frameworks like React, Angular, Vue
    • Massive community support and job opportunities

    Key Features of JavaScript

    • Event-driven programming
    • Dynamic typing
    • Supports OOP (Object-Oriented Programming)
    • Asynchronous programming using Promises & async/await
    • Runs on every device with a browser

    JavaScript Basics

    JavaScript provides variables, operators, data types, functions, and control statements.

    let name = "Vishal";
console.log("Hello " + name);

function add(a, b) {
  return a + b;
}
console.log(add(5, 3));

    Events in JavaScript

    JavaScript responds to user actions like click, mouseover, keypress, scroll, etc.

    document.getElementById("btn").addEventListener("click", () => {

    OOP in JavaScript

    JavaScript supports classes, objects, inheritance, and encapsulation.

    class Person {
  constructor(name) {
    this.name = name;
  }
  greet() {
    console.log("Hello " + this.name);
  }
}

let user = new Person("Vishal");
user.greet();

    Asynchronous JavaScript

    JavaScript supports asynchronous programming using callbacks, promises, and async/await. 

    async function fetchData() {
  let res = await fetch("https://api.example.com/data");
  let data = await res.json();
  console.log(data);
}
fetchData();

    JavaScript Data Types

    JavaScript provides different data types to store values. They are divided into Primitive and Non-Primitive.

    Primitive Types:

    • String
    • Number
    • Boolean
    • Undefined
    • Null
    • BigInt
    • Symbol
    let name = "Vishal";
let age = 21;
let isActive = true;

    JavaScript Operators

    Operators are used to perform operations on variables and values.

    • Arithmetic Operators (+, -, *, /)
    • Assignment Operators (=, +=, -=)
    • Comparison Operators (==, ===, >, <)
    • Logical Operators (&&, ||, !)
    • Ternary Operator
    let x = 10;
let y = 20;
console.log(x + y);
console.log(x > y ? "X is big" : "Y is big");

    JavaScript Arrays

    Arrays are used to store multiple values in a single variable.

    let fruits = ["Apple", "Banana", "Mango"];

fruits.push("Orange");
console.log(fruits);

    Array Methods:

    • push() / pop()
    • shift() / unshift()
    • map()
    • filter()
    • reduce()

    Functions in JavaScript

    Functions are blocks of code designed to perform a specific task.

    // Normal function
function greet() {
  return "Hello Vishal!";
}

// Arrow function
const greet2 = () => "Hello from Arrow Function!";

console.log(greet());
console.log(greet2());

    JavaScript Objects

    Objects store data in key-value pairs.

    const student = {
  name: "Vishal",
  age: 21,
  course: "BCA",
  greet() {
    console.log("Hello, " + this.name);
  }
};

student.greet();

    Modern JavaScript (ES6+)

    ES6 introduced powerful features to simplify coding.

    • let, const
    • Template Literals (`${value}`)
    • Destructuring
    • Spread and Rest operators (...)
    • Classes
    const { name, age } = student;
console.log(name, age);

const nums = [1, 2, 3];
const newNums = [...nums, 4, 5];
console.log(newNums);

    Local Storage

    LocalStorage stores data inside the browser permanently (until deleted).

    // Save data
localStorage.setItem("username", "Vishal");

// Get data
console.log(localStorage.getItem("username"));

// Remove data
localStorage.removeItem("username");

    Fetch API

    fetch("https://jsonplaceholder.typicode.com/posts")
  .then(res => res.json())
  .then(data => console.log(data));
